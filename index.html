Trading Execution Panel for MT5

Oichi Equipment provides a desktop-based trading support panel for MetaTrader 5.
This software allows manual trade execution and position management while using external chart analysis tools.

Features

• Manual order execution (Buy / Sell)
• Position and pending order management
• One-click full close functionality
• Real-time account information display

System Overview

The software operates locally on the user's machine and connects to an active MT5 terminal.
No cloud execution or automated trading logic is included.

Important Notice

This product does not provide financial advice, trading signals, or automated trading.
All trading decisions must be made solely at the user's own responsibility.

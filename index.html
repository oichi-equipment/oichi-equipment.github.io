Trading Execution Panel for MT5

Oichi Equipment provides a manual trading support tool for MetaTrader 5.
This software allows users to execute and manage trades while analyzing charts externally.

Features

• Manual order execution (Buy / Sell)
• Position & pending order management
• Full close functionality
• Account information display

Usage

Runs locally and connects to an active MT5 terminal.
No cloud processing or automated execution is performed.

Important Notice

This product does NOT provide trading signals, automated trading, or financial advice.
All decisions are made at the user's own responsibility.
